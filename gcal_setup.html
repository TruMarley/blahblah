1
Enable Google Calendar API in Google Cloud Console
▶

You need to create a Google Cloud project and enable the Calendar API. This takes about 3 minutes.

  • 1Go to console.cloud.google.com and sign in with your Google account
  • 2Click "New Project" → name it "RingDesk Calendar" → Create
  • 3In the search bar, type "Google Calendar API" and click it
  • 4Click the blue "Enable" button
🖥️
Google Cloud Console — API Library
2
Create credentials and download credentials.json
▶

Create a Service Account — this lets RingDesk write to your calendar without needing your password.

  • 1In Google Cloud Console → IAM & Admin → Service Accounts
  • 2Click "Create Service Account" → name it "ringdesk-cal" → Done
  • 3Click the service account → Keys tab → Add Key → JSON
  • 4A file called credentials.json will download automatically
  • 5In Google Calendar → Settings → Share with: paste the service account email, give editor access
🔑
Service Account Keys page — download JSON

The service account email looks like: ringdesk-cal@your-project.iam.gserviceaccount.com

3

▶

Upload the file you just downloaded. RingDesk stores it securely and uses it to write calendar events.
